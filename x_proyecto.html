Cifrado César

Lenguajes utilizados: HTML, CSS, JavaScript
Unidad Academica: Programación
Especialidad:Programacion
Tema: Seguridad
Delimitación del Tema: Seguridad Web
Propósito: Crear un programa que cifre textos planos
Impacto social: Crear conciensa sobre el uso de la informacion en las redes sociales
Resumen: El usuario ingresa un texto y el programa hace el cifrado y descifrado
Introduccion: Se nos pidio que hicieramos un cifrado con javascript
Objetivos:
General:
-Crear un programa innovador, y utilizar el cifrado Cesar
Particulares:
-Analizar de forma mas concreta este cifrado
-Observar ventajas y desventajas del mismo
Justificación: Pues queriamos hacer un programa que cifrara Cesar, para ejemplificar en clase
Sustento Teórico: el gran conquistador Cesar usaba este codigo para cifrar sus mensajes de guerra y amor :3
Cronograma de Actividades:
Primero se hizo una explicacion sobre los cifrados cesar, y despues se paso a cifrar en clase
Factibilidad Técnica: Pues buscamos que sea lo mas claro posible
Factibilidad Financiera: Pues como somos pobres, usamos herramientas con las cuales ya contabamos de semestres anteriores
Innovación: Pues se uso otro css con el cual quedo bonito :3
Conclusión: Que muy claro para el grupo, y se llevaron de tarea un cifrado a casa.
Sustentabilidad: Pues lo unico que usamos fue sudor, tabaco y muchas horas de nuestras vidas :3
Funcionalidad: completada
Factibilidad: Como antes mencionado, se reutilizaron herramientas de anteriores cursos
Autor/es: Juan, Pablo, Javier
Grupo: 5IV7
